# Tools and Helper Scripts

## build-podman.sh

Builds the Bookmarks Manager container image using Podman.

```bash
./build-podman.sh
```

## assign-iam-roles.sh

Assign Google Cloud IAM roles to a user at project level.

```bash
./assign-iam-roles.sh [PROJECT_ID] [GOOGLE_ACCOUNT_EMAIL]
```

## download-bootstrap.sh

Download Bootstrap CSS and Icons to static folder.

```bash
./download-bootstrap.sh
```
